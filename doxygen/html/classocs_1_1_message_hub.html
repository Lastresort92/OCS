Detailed Description

Handles the posting and retrieving of messages. Users called "Transceivers" may create an object that inheritys from "Message", and post this object for other transceivers to see. These messages should be cleared periodically. Registered Transceivers also receive a message slot that other transceivers can send messages to. These messages stay active until the recipient transceiver views these messages.

Messages are sorted by their type. There is no need for a transceiver to subscribe to a certain message type. They can simply get a list of the desired message type from the message board.
