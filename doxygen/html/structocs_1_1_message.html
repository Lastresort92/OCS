template<typename T>
struct ocs::Message< T >

A user defined message can be created by inheriting from "Message" and passing in the type as the template paramater. e.g. struct TextMessage : public ocs::Message<TextMessage> { TextMessage(const Transceiver& _transceiver, const std::string& _msg) : Message(_transceiver), msg(_msg) {.
